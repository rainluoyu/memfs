# MemFS - Memory-First File System

## 核心依赖
# 标准库已包含所有核心功能，以下是可选依赖

# 压缩算法（可选）
lz4 >= 4.0.0          # 快速压缩
zstandard >= 1.5.0    # 高压缩率

# 系统监控（可选，推荐）
psutil >= 5.9.0       # 内存和磁盘监控

# 测试
pytest >= 7.0.0       # 测试框架
pytest-cov >= 4.0.0   # 覆盖率

# 开发
black >= 23.0.0       # 代码格式化
mypy >= 1.0.0         # 类型检查
